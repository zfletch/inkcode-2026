# Applications Created for InkCode 2026

## Acrostic Finder

Tool for working with ancient Greek acrostics in the
*Anthologia Palatina*.
See https://github.com/zfletch/onymatic for more information.

## Coffee Compass and Arnth's Lament

These were created with a local model as experiments to test
and demonstrate their capabilities. The model is
(Gemma 4)[https://deepmind.google/models/gemma/gemma-4/]
on Ollama with the following capabilities:

```
  Model
    architecture        gemma4
    parameters          8.0B
    context length      131072
    embedding length    2560
    quantization        Q4_K_M
    requires            0.20.0
```
